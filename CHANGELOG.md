# Changelog

## [1.0.0] - 2026-06-22

### Added
- Initial release aplikasi Surat Perintah
- User authentication dengan JWT
- CRUD operations untuk Surat Perintah
- Auto-generate nomor surat
- Preview dan export PDF
- Approval workflow
- PostgreSQL database dengan migrations
- React frontend dengan TypeScript
- Express backend dengan TypeScript
- Docker support
- Zustand state management
- Tailwind CSS styling

### Features
- Register dan Login user
- Buat surat perintah dengan form dinamis
- Edit draft surat
- Preview surat sebelum save
- Export surat ke PDF
- Approval workflow untuk surat
- History surat dengan pagination
- Role-based access control (admin, creator, approver)
- Responsive design
